OSCEtimer.app

The OSCE Timer is a free, web-based synchronized countdown timer based on peerJS.

All users are connected to a synchronized timer via a peer-to-peer connection using the ID provided by the administrator.

An administrator controls the countdown timer (start / stop / countdown duration). A listener listens to the time signal broadcast from the administrator. A peer-to-peer connection ensures that the timers on different connected devices are synchronized.
